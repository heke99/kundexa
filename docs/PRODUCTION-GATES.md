# Kundexa production gates

The application code is not the sole production approval. These gates must be closed with documented owners and evidence:

1. Signed commercial and data-processing arrangement with 46elks, including subaccounts, usage limits, resale, incident handling and credit liability.
2. Written confirmation of outbound WebRTC/SIP configuration for Kundexa's exact call flow.
3. Number allocation, porting, emergency-service and geographic-number obligations per market.
4. Tenant billing, prepaid credit, cost caps, failed payment and abuse allocation.
5. Swedish legal review of every B2C phone-sales template and acceptance flow.
6. Sector review for electricity, telecom, insurance, finance and any regulated vertical.
7. Recording legal basis, customer notice, DPA, access model, retention and deletion process.
8. Final trademark, company-name and domain clearance for Kundexa.
9. Independent penetration test and RLS isolation test against the deployed environment.
10. Backup restore exercise, incident runbook, key rotation and disaster recovery sign-off.
